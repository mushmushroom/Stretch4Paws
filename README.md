# Stretch4Paws
A simple and interactive one-page web application for desk-friendly stretching routines.

## Features
- Individual stretch blocks with per-stretch timers
- Global progress bar for the full routine
- Start/Pause/Reset functionality
- Lottie animations for each stretch
- Confetti/celebration animation when routine is completed
- Dark and light theme toggle

## Tech Stack
- Frontend: Vite + React + TypeScript
- Styling: SCSS with CSS variables for theming
- Animations: Lottie React with custom animation made in AfterEffects
- Icons: React Icons
- State Management: React Context + Hooks

## Preview
To view the deployed version, visit [this link](https://stretch4-paws.vercel.app/)

## How to run this project
1. Clone the repository
``` 
git clone https://github.com/mushmushroom/Stretch4Paws.git
```
2. Install dependencies
```
npm install
```
3. Start the development server
``` 
npm run dev
```
4. Open the app in your browser
```
http://localhost:5173
```

## Notes
- You can add new stretches easily by editing stretches.ts and supplying corresponding Lottie frames
